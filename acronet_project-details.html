Technical Stack:

Acronet is built upon a modern and efficient technical stack, combining the following technologies and tools:

Frontend:

Angular:

Utilized for developing the frontend interface, Angular provides a robust framework for building dynamic and responsive web applications.

TypeScript:

Leveraging TypeScript enhances the development process by enabling static typing, improved code organization, and better IDE support.

Backend:

Django:

Employed as the backend framework, Django offers a powerful and secure foundation for building web applications, providing features such as ORM, authentication, and routing.

SQL Database:

Utilized for data storage and retrieval, SQL ensures efficient management of relational data within the Acronet platform.

Additional Tools:

Docker:

Docker was utilized to containerize the application using Docker images, streamlining the deployment process and ensuring consistency across different environments. This approach enhances scalability, portability, and ease of maintenance for the Acronet platform.

Acronet

Acronet is a comprehensive software solution developed as part of a software engineering course to cater to the diverse needs of our university community regarding commonly used acronyms. Designed with user convenience in mind, Acronet offers a robust search functionality along with innovative features such as a Recent Search tab for quick access to previously searched acronyms and a New Acronyms tab to keep users informed about the latest additions. Additionally, the platform includes an administrative panel enabling authorized personnel to review, approve, reject, or edit suggested acronyms submitted by registered users. Moreover, the Manage Users tab facilitates the seamless management of user roles, allowing designated individuals to grant administrative privileges for streamlined approval processes.
