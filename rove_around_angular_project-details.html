Technical Stack:

Rove Around leverages a robust technical stack to deliver a seamless and feature-rich user experience:

Frontend:

Angular:

Utilized for frontend development, Angular provides a scalable and modular framework for building dynamic web applications.

TypeScript:

TypeScript enhances the development process by offering static typing, improved code organization, and better IDE support, ensuring robust frontend functionality.

Backend:

Java Spring Boot:

Employed as the backend framework, Java Spring Boot offers a lightweight and efficient solution for building RESTful APIs, providing features such as dependency injection, MVC architecture, and security.

MySQL Database:

Utilized for data storage and retrieval, MySQL offers a reliable and scalable solution for managing relational data within the Rove Around application.

Additional Tools:

Google Maps:

Integrated to visually represent selected destinations via markers, Google Maps offers an intuitive and interactive mapping solution, enhancing the planning and visualization capabilities of Rove Around.

Google Autocomplete API

Integrated to facilitate destination selection, the Google Autocomplete API enhances the trip planning process by providing real-time suggestions based on user input, improving accuracy and efficiency.

Serp API - Google Real Time Search Engine

Utilized as the Google real-time search engine for places results, the SERP API enhances the application by providing up-to-date information on places, enriching the user experience and ensuring accuracy in trip planning.

Rove Around Trip Planner

Rove Around is an innovative trip planning application developed as part of a capstone course, aimed at facilitating seamless travel organization among friends. Led by a team with a focus on user-centric design, Rove Around offers a comprehensive suite of features to streamline trip planning and management. Users can outline trip plans with ease, leveraging Google Autocomplete API for efficient destination selection. Integration with Google Maps allows for visual representation of selected destinations via markers, enhancing planning accuracy and user experience. Furthermore, the platform enables users to add notes, plan itinerary day-wise, manage budgets and expenses, and conveniently split costs among trip mates. Additionally, Rove Around offers a dashboard to track the status of trips, whether completed, ongoing, or upcoming, providing users with a centralized hub for trip management.
